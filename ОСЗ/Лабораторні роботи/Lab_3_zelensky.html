import cv2 as cv
import math
import matplotlib.pyplot as plt
import os

def resize_proportional(image, k_width):
    (height_N, width_M) = image.shape[:2]
    ratio = k_width / float(width_M)
    new_height = int(height_N * ratio)
    new_dimensions = (k_width, new_height)
    resized_image = cv.resize(image, new_dimensions, interpolation=cv.INTER_AREA)
    
    return resized_image

Контрольні запитання

  1. У чому полягає сенс афінних перетворень?

    2. Сенс афінних перетворень полягає у збереженні паралельності. Це означає, що будь-які лінії, які були паралельними на початковому зображенні, залишаться паралельними і після трансформації. Ці перетворення є комбінацією лінійних операцій (масштабування, поворот, зсув) та трансляції (переміщення). Хоча вони зберігають прямі лінії, кути та відстані між точками можуть змінюватися.

  2. Чи є оперції трансляції, масштабування і повороту афінними? Відповідь обгрунтувати.

    3. Так, ці операції є окремими випадками афінних перетворень.

    Кожна з них підпорядковується головному правилу афінних перетворень — збереженню паралельності:

    • Трансляція (зсув): Лише переміщує зображення; всі лінії зберігають свій кут нахилу та паралельність.
    • Масштабування (зміна розміру): Збільшує або зменшує зображення, але паралельні лінії залишаються паралельними.
    • Поворот: Обертає всі лінії на однаковий кут навколо центру, тому лінії, що були паралельними, залишаються паралельними і після повороту.
  3. Чи є перспективні перетворення частинним випадком афінних перетворень? Відповідь обгрунтувати.

    4. Ні, це не так. Насправді, афінне перетворення є більш обмеженим, частковим випадком перспективного.

    • Афінні перетворення (на основі 3 точок) завжди зберігають паралельність ліній.
    • Перспективні перетворення (на основі 4 точок) не зберігають паралельність. Вони можуть змусити паралельні лінії “сходитися” в одній точці, імітуючи 3D-перспективу. Це дозволяє перетворити трапецію на прямокутник, що неможливо зробити лише афінним перетворенням.
