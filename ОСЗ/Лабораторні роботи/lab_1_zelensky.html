Контрольні запитання

  1. Що таке pip і conda та чим вони відрізняються між собою?

    2. pip (Pip Installs Packages) — це офіційний менеджер пакетів для Python. Він встановлює пакети (бібліотеки) з репозиторію PyPI (Python Package Index). pip керує лише пакетами Python і для керування віртуальними середовищами потребує додаткових інструментів, як-от venv або virtualenv.

    conda — це одночасно менеджер пакетів та менеджер віртуальних середовищ. Він є частиною дистрибутиву Anaconda/Miniconda і встановлює пакети з репозиторіїв Anaconda. Головна відмінність conda в тому, що він може керувати пакетами для будь-яких мов (не лише Python, але й R, C++ тощо) та їхніми складними залежностями. conda також самостійно створює та керує ізольованими середовищами (conda create, conda activate).

  2. Як буде виглядати виконання пунктів 1-3 індивідуального завдання за допомогою Virtualenv?

    3. Спочатку створюємо віртуальне середовище командою: python -m venv myenv

    Потім активуємо його: myenv/Scripts/activate

    Встановити пакети: pip install numpy matplotlib opencv-python jupyter

    Запустити Jupyter

  3. Як виконати візуалізацію зображення (див. п.3) засобами mathplotlib?

    4. Для коректної візуалізації зображення, завантаженого через OpenCV, за допомогою matplotlib, необхідно конвертувати його колірний простір. OpenCV (cv.imread) завантажує зображення у форматі BGR (Синій-Зелений-Червоний), тоді як matplotlib (plt.imshow) очікує формат RGB (Червоний-Зелений-Синій). 

    
import cv2 as cv
import matplotlib.pyplot as plt

# 1. Завантажити зображення (буде в BGR)
image_bgr = cv.imread('шлях/до/вашого/зображення.jpg')

# 2. Конвертувати з BGR в RGB
image_rgb = cv.cvtColor(image_bgr, cv.COLOR_BGR2RGB)

# 3. Відобразити за допомогою matplotlib
plt.imshow(image_rgb)
plt.title('Кольорове зображення')
plt.axis('off') # Вимкнути осі
plt.show()
